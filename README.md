# 📊 Causal Impact Analysis of Pricing Strategy

## 🎯 Objective
To evaluate whether a discount strategy actually caused an increase in sales using statistical modeling.

---

## 🧠 Problem Statement
Businesses often assume discounts increase sales.  
This project analyzes whether the increase is causal or coincidental.

---

## ⚙️ Approach
- Created simulated sales dataset  
- Defined treatment variable (before vs after discount)  
- Applied regression model using Python  
- Measured causal impact  

---

## 📊 Result
✅ Discount increased sales by approximately **43.5 units**

---

## 🛠️ Tools Used
- Python  
- Pandas  
- NumPy  
- Statsmodels
